## 📖 知识库

### 黄金交易核心指标
| 指标 | 含义 | 用法 |
|------|------|------|
| DXY | 美元强弱 | DXY涨→金价跌 |
| 美债收益率 | 利率预期 | 收益率涨→金价承压 |
| 原油 | 通胀预期 | 油价涨→金价涨 |
| VIX | 恐慌指数 | VIX涨→避险→金价涨 |
| CFTC持仓 | 资金流向 | 净多头增加→看涨 |

### 交易时段（北京时间）
| 时段 | 时间 | 特点 |
|------|------|------|
| 亚盘 | 07:00-15:00 | 波动小，震荡为主 |
| 欧盘 | 15:00-20:30 | 波动增大 |
| 美盘 | 20:30-03:00 | 波动最大，数据行情 |

### 风险管理
- 单笔亏损 ≤ 2%
- 盈亏比 ≥ 1:1.5
- 凯利公式仓位管理
- 日内最多3笔交易
